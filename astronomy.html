Describes a reusable suite of sky navigation tools. Notably:

  • “Digital Pair” Telescope
  • Field Rendering
  • Listing and finding targets

 

Telescope

A “digital pair” object to represent the state of the real-world scope. This keeps track of number of parameters such as key telescope measurements (aperture, focal length, eyepiece, apparent field of view), current position (right ascension/declination) and location (latitude/longitude), and camera offset (two float values). In addition, the class contains a few methods for getting the current position of the scope, taking into account the offset, and getting magnification and limiting magnitude. 

Field Rendering

Starplot is used for rendering images of telescope's current field of view. The telescope object is used directly for its parameters to generate the most accurate representation.  

Target Manager

Targets will be loaded from a few different catalogs, such as Messier and NGC. The location of other targets can be acquired through Skyfield, and be set as a target position. Using the previously mentioned telescope object, we can use its position to easily calculate a distance and angle to the target. For the UI, this information can be used to draw an arrow out from the center of the field, with its length representing the distance away.
