This website documents the creation of an astronomy tool that utilizes a camera for plate solving to actively track a telescope's position for dynamic sky navigation.

 

To get started, you can review Project Components and Requirements.

To more easily understand the system, it will be broken up into three core component groups:

  • Image Processing Pipeline (controlling capture, solver)
  • Build and Hardware Implementation (screen and input)
  • Astronomy Tools (target management and rendering)

This will allow us to create a modular, hardware-agnostic tool that will be easy to extend.
